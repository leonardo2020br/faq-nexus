---
question: "Como configurar a integração com a API do Itaú (boleto híbrido)?"
category: financeiro
search_keywords: "itau api boleto hibrido certificado integracao banco"
order: 20
---

A integração exige a geração e instalação de um certificado digital no portal do Itaú. O processo envolve: gerar o certificado no portal Itaú Developers, instalar no servidor e configurar os dados da carteira no Nexus.

Consulte o passo a passo completo: [Certificados API Itaú (Boleto Híbrido)](/artigos/itau-certificado-api/)
