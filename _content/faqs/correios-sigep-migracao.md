---
question: "Como migrar a integração dos Correios do SIGEP Web para a API?"
category: suporte
search_keywords: "correios sigep api migracao integracao etiqueta rastreio"
order: 110
---

A migração tem 3 etapas: (1) alterar o **Modo** para *Sigep Via API* em Configurações → Empresas → Aba 17-Integrações, (2) gerar o **Token de API** no portal dos Correios em `cws.correios.com.br`, (3) atualizar o cadastro da transportadora CORREIOS com o novo token.

Guia completo: [Migração Correios: SIGEP Web → API](/artigos/correios-api-integracao/)
