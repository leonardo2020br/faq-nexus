---
question: "A integração com os Correios parou de funcionar. O que verificar?"
category: suporte
search_keywords: "correios api token expirou integracao parou etiqueta erro"
order: 112
---

O motivo mais comum é o **Token de API expirado**. Acesse o portal dos Correios em `cws.correios.com.br`, gere um novo token e atualize a **Senha** no cadastro da transportadora CORREIOS em Cadastros → Parceiros de Negócios → aba Transportadora.
